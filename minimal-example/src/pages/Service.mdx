# Overview

This service is responsible for the following:

- Say Hello World whenever a request is made to its /hello API endpoint

# Change Process

## Deployment

Run the following to install:

```bash
npm install
npm start
```

## Validation

- Check Service Health
  ```shell
  curl http://{{ service_fqdn }}/api/v2/hello
  ````
